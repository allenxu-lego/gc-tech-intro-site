# Product Steering

## Overview

Tapestry GC Technology Enterprise Portal - an internal technology showcase and documentation site for the company's DevOps platform capabilities.

## Purpose

Provide a centralized, interactive visualization of the organization's CI/CD DevOps workflow, enabling employees and stakeholders to understand the software delivery pipeline.

## Target Users

- Internal employees learning about DevOps processes
- Stakeholders reviewing technology capabilities
- New team members onboarding to the engineering organization

## Key Features

1. **Interactive CI/CD Flowchart**
   - Visual representation of the DevOps pipeline
   - Step-by-step workflow animation (Developer → GitLab → Jenkins → Production)
   - Execution steps visualization for Jenkins pipelines
   - Dual pipeline paths: Service deployment and Dify (AI platform) deployment

2. **Enterprise Portal Structure**
   - Collapsible sidebar navigation
   - Consistent header with brand identity
   - Responsive layout with yellow brand accent

## Brand Identity

- **Name**: Tapestry GC Technology
- **Primary Color**: Yellow (#EAB308)
- **Background**: White (#FFFFFF)
- **Design Style**: Minimalist, clean, professional

## Success Metrics

- Clear understanding of DevOps workflow through interactive visualization
- Successful static export capability for offline access
- Consistent brand presentation across all pages

## Future Considerations

- Additional workflow visualizations
- Integration with actual CI/CD status data
- Multi-language support
